# Project Memory

## Core
Safari booking system for BOGA (Botswana Guides Association). DM Sans body, Outfit headings.
Warm safari donkey-brown palette — no pure white backgrounds. Amber/gold accents.
Lovable Cloud backend. Staff dashboard at /admin (called "Reservation" in UI), entered by clicking BOGA in footer.
Logins: admin@boga.org.bw / BogaAdmin2024! · accountant@boga.org.bw / BogaAccounts2024!
User prefers modern, clean design — not old-fashioned. No green color themes.
BOGA Reserve Camp (Maun) = individuals only, per-person × nights pricing (P250 pp/night). Other parks = companies/members.
