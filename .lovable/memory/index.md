# Project Memory

## Core
Safari booking system for BOGA (Botswana Guides Association). DM Sans body, Outfit headings.
Warm safari donkey-brown palette — no pure white backgrounds. Amber/gold accents.
Lovable Cloud backend with Supabase. Admin route at /admin with Supabase auth.
Admin login: admin@boga.org.bw / BogaAdmin2024!
User prefers modern, clean design — not old-fashioned. No green color themes.
Documents called "vouchers" not "invoices". Site voucher codes hidden until confirmed.
Footer credit: "Made with ♥ by Pamoja" (purple, links to pamojadigital.org).
